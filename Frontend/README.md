# NGO Impact Optimizer Frontend

## Setup

1. Install dependencies:

   ```powershell
   cd Frontend
   npm install
   ```

2. Start the development server:

   ```powershell
   npm run dev
   ```

3. Open the app in your browser at `http://localhost:5173`.

## Notes

- The frontend connects to the backend API at `http://localhost:5000`.
- Tailwind CSS is configured with `tailwind.config.js`.
- Recharts powers the dashboard visualizations.
